## How to get back to where we left off

Anvil state resets on restart. To redeploy:

```bash
# Terminal 1 — start Anvil
cd ~/madness-app/bracket-pool-dapp/contracts
export PATH="$HOME/.foundry/bin:$PATH"
anvil

# Terminal 2 — deploy MockUSDC + Factory
cd ~/madness-app/bracket-pool-dapp/contracts
export PATH="$HOME/.foundry/bin:$PATH"
PK0=0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80

# Deploy MockUSDC
forge create test/mocks/MockUSDC.sol:MockUSDC \
  --rpc-url http://127.0.0.1:8545 --private-key $PK0 --broadcast
# → MockUSDC at 0x5FbDB2315678afecb367f032d93F642f64180aa3

# Set .env and deploy factory
cat > .env << 'ENVEOF'
PRIVATE_KEY=0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
USDC_ADDRESS=0x5FbDB2315678afecb367f032d93F642f64180aa3
TREASURY_ADDRESS=0x70997970C51812dc3A010C7d01b50e0d17dc79C8
ENVEOF

source .env
forge script script/Deploy.s.sol:DeployScript \
  --rpc-url http://127.0.0.1:8545 --private-key $PK0 --broadcast
```

## Reference: Key Design Decisions

See `docs/phase-0-decisions.md` for full details. The most important:

- **67 games, 7 rounds** — scoring: R0=5, R1=10, R2=20, R3=40, R4=80, R5=160, R6=320
- **Hash-only storage** — contract stores `keccak256(picks)`, full picks emitted in events
- **Fee formula** — `totalPoolValue * 500 / 10000` (5%), scorer must match integer division
- **Claim deadline** — `finalizeDeadline + 90 days`, then admin can sweep unclaimed
- **Tiebreaker** — predicted championship total score, closest wins, ties split evenly
